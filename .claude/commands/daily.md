# Daily Note作成

今日のデイリーノートを作成してください。

## 設定
- 保存先: `00_Development_Notes/ai-trade-pj/daily_notes/`
- ファイル名: `YYYY-MM-DD.md`（今日の日付）
- テンプレート: `04_Templates/Daily.md`

## 手順
1. Obsidian MCPツールでテンプレートを取得
2. 日付変数を今日の日付に置換
3. デイリーノートを作成
4. 既に存在する場合は作成せず、その旨を伝える
